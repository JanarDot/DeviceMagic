# Device Magic Site

This repository is the public marketing site for Device Magic.

It is intentionally separate from the private `DeviceMagic` distribution repo.

## Purpose

- host the landing page
- support GitHub Pages publishing
- keep public site content separate from private app and release work

## Files

- `index.html`
- `styles.css`

## Local Preview

Open `index.html` in a browser or serve the folder with a simple static server.

## Notes

- this repo is meant to be public
- the app/distribution repo remains private
